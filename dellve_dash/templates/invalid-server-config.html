Unprocessable Request: Invalid Server Configuration Provided



Dellve Dash could not process the provided server configuration due to one of the following:

  • Request missing required config params ( dellve_port, netdata_port, server_ip )

  • Given config params missing required dependencies

    • Dellve API not installed/visible

    • Netdata or nvidia profiler pluggin not installed/visible



To access Dellve Dash, please enter a valid Dellve server configuration in the Portal Home

For information on how to configure your server to be compatible with Dellve Dash, visit the Dellve Dash READme on Github
